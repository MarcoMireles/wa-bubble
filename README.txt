== Description ==
Add a whatsapp bubble so your users can contact you directly faster and more efficiently.
With WA Bubble you won\'t have to worry about those long and complicated installations since it is very easy to use. WA Bubble does not install unnecessary things for you, it is a simple plugin that covers your need to stay connected with your users.

== Installation ==
Go to the plugins section
Search or upload the WA Bubble plugin
Activate and Ready
